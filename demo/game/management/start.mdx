POST `game/management/start`

Starts a new game

:::info Note

This is a privileged endpoint. Only authorised queries will be processed.

:::

## Parameters

`name` *string*  
Game name

`max_players` *integer*  
Number of players that can join the game.

`can_fly` **optional** *boolean*  
If players can fly in this game. Default False.
