GET `game/stats/win_loss/$player`

Returns player's Win/Loss statistics.

:::info Note

This is a privileged endpoint. Only authorised queries will be processed.

:::

## Parameters

`player` **optional** *string*  
The UUID of the player. If none supplier, return all players.
