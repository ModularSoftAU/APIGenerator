POST `player/edit_health`

Adds the specified amount to the player's health. Can be negative."

:::info Note

This is a privileged endpoint. Only authorised queries will be processed.

:::

## Parameters

`player` *string*  
The UUID of the player.

`amount` *integer*  
Health to add (can be negative)
